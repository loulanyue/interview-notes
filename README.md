## :memo: 面试复习建议

下面整理按主题推进复习时的一些小提示，方便在准备八股、查漏补缺和阶段性回顾时快速建立节奏。

- 2023年1月1日：围绕操作系统补充一条复习建议，建议先建立目录感，再回到知识点细节。这更适合第一轮打基础。
- 2023年1月2日：围绕网络补充一条复习建议，可以先过一遍核心概念，再挑典型问题做口述练习。这对中后期查漏补缺很有帮助。
- 2023年1月3日：围绕面向对象补充一条复习建议，复习时优先整理高频问法，比平铺直叙更适合面试场景。这样更方便在面试时快速组织答案。
- 2023年1月4日：围绕数据库补充一条复习建议，把概念、优缺点和适用场景放在一起记忆，会更容易输出。这能帮助你把零散知识连成体系。
- 2023年1月5日：围绕Java补充一条复习建议，阶段性回顾时，尽量把零散知识串成完整问题链路。这会让复习节奏更稳定一些。
