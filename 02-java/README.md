# 02 Java

Java 主题按内容拆分为以下几个子目录：

- `basics/`：基础语法、面向对象、常见语法与日常开发片段
- `collections/`：集合框架与常见容器问题
- `concurrency/`：并发、多线程、线程池
- `jvm/`：JVM、内存模型、GC、诊断工具
- `io/`：IO、文件与流
- `new-features/`：Java 8/14 等新特性
